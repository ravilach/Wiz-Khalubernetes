Wiz Khalubernetes API Documentation

POST /api/quotes
Submit a new Wiz Khalifa quote
{ "quote": "string" }
Response: 
{ "id": 1, "quote": "string", "timestamp": "2025-08-19T...", "ip": "...", "quoteNumber": 1 }
GET /api/quotes
Get all quotes
Response: 
[ { "id": 1, "quote": "string", ... }, ... ]
DELETE /api/quotes/{id}
Delete a quote by ID
Response: 
"Deleted"
GET /api/quotes/latest
Get the latest quote
Response: 
{ "id": 1, "quote": "string", ... }
GET /api/nodeinfo
Get node/system/application info
Response: 
{ "hostname": "...", "app": "...", ... }
GET /api/dbstatus
Get current DB connection status/type
Response: 
{ "type": "H2", "connected": "true", "message": "Connected to H2" }
